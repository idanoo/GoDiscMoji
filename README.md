# GoDiscMoji
Discord bot to measure emoji usage
    

### Usage
- Copy .env.example to .env and fill in DISCORD_TOKEN from your bot
- `docker compose up -d`
    

### Commands
```
/show-top-users
# Shows top 5 emoji users and their top 3 emojis

/show-top-emojis
# Shows top 5 emojis and their 3 biggest users
```
